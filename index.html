Usage

U is the universal set. It is the set of all possible elements.

A is a set. It is a subset of the universal set.

Equation is the equation to be evaluated. It can contain the following operators:

  • V union
  • ^ intersection
  • ' complement (can only be used if U is present)

Example: (A V B) ^ C' (case sensitive)
